Budget Tracker (Python CLI App)

A simple but functional command-line Budget Tracker built in Python.
This application allows users to set an initial budget, add expenses, view spending summaries, and automatically save/load data using a JSON file.

Designed as a beginner-to-intermediate project to practice file handling, data structures, functions, loops, and basic persistence.
